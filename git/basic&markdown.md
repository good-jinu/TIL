# GIT

## 1. 개요

> git은 분산 버전 관리 프로그램

## 2. 설치 및 실행

- git 공식 사이트에서 설치가능
- bash셸로 실행

## 3. bash셸 명령어

- ls: 디렉토리 탐색
- cd: 디렉토리 이동
- mkdir: 디렉토리 만들기

## 4. vscode 연동

- vscode 터미널을 git bash셸로 바꿈
- vscode 내에서 git bash를 그대로 사용가능
